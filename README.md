# Memory Matching Game

A simple and fun **Memory Card Matching Game** built with **HTML, CSS, and JavaScript**.  
Flip cards, match pairs, and try to complete the game with the best time / fewest moves.

## Live Demo
- Deployed: `https://memory-matching-game-beta.vercel.app/`

---

## Features
- Interactive **flip-card** gameplay
- Match logic with **pair detection**
- Simple UI with responsive styling
- Restart / play again functionality *(if included in your game)*
- Lightweight and fast (no frameworks)

---

## Tech Stack
- **HTML**
- **CSS**
- **JavaScript (Vanilla JS)**

## How to Run Locally
### Option 1: Open directly
1. Download the repo
2. Open `index.html` in your browser

### Option 2: VS Code Live Server (recommended)
1. Open folder in VS Code
2. Install **Live Server**
3. Right-click `index.html` → **Open with Live Server**
